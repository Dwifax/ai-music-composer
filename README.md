# AI Music Composer

Algorithmic music composition system for generating melodies, chord progressions, and arrangements.

## Features
- Scale system: major, minor, pentatonic, blues, dorian
- Melody generation with weighted random walks
- Chord progressions: pop, jazz, epic, sad templates
- Text-based score rendering

## Usage
```bash
python3 composer.py [root_midi] [scale] [style]
python3 composer.py 60 minor epic
```
